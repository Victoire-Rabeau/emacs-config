# emacs_config

This is my emacs config for Epitech.

Commands:
  C-s: save
  C-q: leave
  C-t: treemacs
  C-e: treemacs add project
  C-r: treemacs remove project
  C-s: save
  C-z: undo
  C-c: copy
  C-x: cut
  C-v: paste
  C-H: Epitech header
  M-S-up: move selection up
  M-S-down: move selection down
